## The aim:
The aim here is to fill out some files:
- angus-burger.md
- veggie-burger.md
- mushroom-burger.md
- blue-cheese-burger.md

Each file should contain:
```md
top bun
> list of ingredients, line by line
bottom bun
```

An example burger might look like:
```md
top bun
lettuce
tomato
cheese
onion
burger patty
pickles
bottom bun
```

## The challenge rules:

1. You can only add one ingredient or file per commit
2. You must push after each commit
3. Ensure that your ingredient is in the `main` branch before adding another ingredient


## Notes
1. Not all of the burgers have been added to the repository yet!
2. Files must be named in `kebab-case`.
3. `main` is protected. You must open Pull Requests to merge into main.
    - Those pull requests must "gain approval" before they can be merged -- you must collaborate.
